# Tokenized Employee Benefits and Incentives Platform

This project uses Clarity on Stacks to implement a token-based system for employee benefits and rewards.

## Prerequisites
- Node.js >=14
- Clarinet CLI
- Git

## Setup
1. `git clone <repo-url>`
2. `cd token-benefits-platform`
3. `npm install`
4. `clarinet test` to run tests

## Structure
- `contracts/` – Clarity smart contracts
- `tests/` – Clarinet tests
- `scripts/` – Deployment & helper scripts
- `Clarinet.toml` – project config

## Usage
- Develop contracts in `contracts/`
- Run `clarinet test` after changes
